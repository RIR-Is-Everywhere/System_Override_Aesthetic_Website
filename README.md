## 🚀 Overview
This "Easy and Simple" project demonstrates how to create a premium, immersive user interface using pure vanilla technologies and high-fidelity libraries. It features:
- **Advanced Preloader**: A simulated terminal crack sequence for the ultimate "Matrix" feel.
- **Three.js Liquid Hologram**: A high-performance, responsive 3D symbiote hero background.
- **GSAP & ScrollTrigger**: Butter-smooth reveals and scroll-bound animations.
- **Vibrant Design System**: A custom CSS theme with neon pink, cyan, yellow, and purple accents.
- **Glassmorphism**: Elite UI components with backdrop blurs and holographic glows.
- **Custom Magnetic Cursor**: Zero-lag, interactive pointer system.

---

## 🛠️ Tech Stack
- **Structure**: Semantic HTML5
- **Style**: [Tailwind CSS](https://tailwindcss.com) (via CDN) & Vanilla CSS3
- **Animations**: [GSAP](https://greensock.com/gsap/) & [Lenis](https://github.com/studio-freight/lenis) (Smooth Scroll)
- **3D Engine**: [Three.js](https://threejs.org)
- **Interactivity**: [Vanilla-Tilt.js](https://micku7zu.github.io/vanilla-tilt.js/) & Font Awesome 6
